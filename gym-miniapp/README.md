# GMS Mini App

`gym-miniapp` 是 Gym Management System (GMS) 的 UniApp 小程序端目录，当前承载会员端与教练端页面骨架和请求逻辑。

## 功能定位

- 会员登录
- 会员首页
- 场馆浏览
- 课程浏览
- 教练浏览
- 我的预约
- 签到核销
- 钱包/资产查看
- 教练审批和工作台页面骨架

## 技术栈

- UniApp
- Vue 单文件组件
- HBuilderX / UniApp 兼容工具链

## 使用方式

当前仓库未内置完整的 UniApp CLI 依赖，建议直接：

1. 使用 HBuilderX 打开 `gym-miniapp`
2. 配置微信小程序 AppID（如需要）
3. 在 HBuilderX 内运行或构建

`package.json` 中保留了基础脚本，主要用于提示标准运行入口。
